Auralog

Your Health • Your Data • Your Way

🌱 Wellness Focus

Auralog is your private health companion. It helps track symptoms, nutrition, sleep, work, weight, nicotine, and medication – while never sharing your data anywhere. Everything is stored only on your device, fully encrypted.

Accessibility is part of Auralog’s DNA: stimming tools, live transcription, and calming themes make it inclusive for neurodivergent and chronic health journeys.

🔒 Privacy & Security

  • All data is encrypted (AES‑256‑GCM) – nothing stored in plaintext.
  • Unlock with PIN (hardened PBKDF2) or Passkey/Biometrics.
  • Automatic lockout after inactivity (default 10 minutes).
  • Encrypted Audit Trail – logs when entries are added, viewed, or exported, supporting HIPAA audit rules.

🛡️ HIPAA Alignment

Auralog is actively built with HIPAA’s Technical Safeguards (§164.312) in mind. Current status:

  • ✅ Access Control – PIN/Passkey auth + auto‑logoff
  • ✅ Integrity – AES‑GCM check + encrypted audit trail
  • ✅ Authentication – PIN + Biometrics
  • ✅ Local Audit Controls – append‑only secure logs
  • 🚧 Pending: Immutable log backups, secure deletion guarantees

*Note: Auralog is not a medical device. It is a self‑tracking and reflection tool to complement professional care.*

🚦 How to Get Started

  1. Install Node.js v18+ and Git on your system.
  2. Clone the repo: git clone …
  3. Run locally with npm install && npm run dev.
  4. First run: app must be served at HTTPS or localhost for install prompt & offline mode to work.

Auralog treats your health data as yours alone – no hidden sync, no cloud store, no brokers.
